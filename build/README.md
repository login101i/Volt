# Build Assets Directory

Place your application icon here:

- `icon.ico` - Windows icon file (256x256 pixels recommended)
- `icon.png` - Alternative PNG format (will be converted if needed)

## Creating an Icon

1. Design your icon (256x256 pixels minimum)
2. Convert to .ico format using:
   - Online tools: https://icoconvert.com/
   - ImageMagick: `magick convert icon.png -resize 256x256 icon.ico`
   - Or use any icon editor

3. Save as `icon.ico` in this directory

The icon will be used in:
- The application window
- The installer
- Desktop shortcut
- Start menu entry
